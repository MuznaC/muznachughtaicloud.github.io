System administration with Python

Accessing the AWS Cloud9 IDE & creating your Python file

In the AWS Management Console, choose Services > Cloud9. In the Your environments panel, locate the reStart-python-cloud9 card, and choose Open IDE.

In AWS Cloud9 environment choose File > New From Template > Python File and save your python file.

In your AWS Cloud9 IDE, choose the + icon and select New Terminal. To display the present working directory, enter pwd

To confirm the default version of Python that is installed in your lab, in the open terminal tab, enter:

python --version

To check other available versions of Python, enter the following commands:

python2 --version
python3 --version

In the navigation pane open previous phython file and paste the following code:

print("Hello, World")

Try more commands and run them in the IDE window.
